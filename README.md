# VIM

## Usage
Go to your home directory, then type the following: 
* ```git clone git@github.com:francisengelmann/vim.git```
* ```mv vim .vim```
* ```ln -s .vim/vimrc .vimrc```

### Snippets
* Ctrl-p
* Ctrl-x-f
